ИНСТРУКЦИЯ ПО СОЗДАНИЮ АРХИВА:

1. Откройте папку C:\Dev\event-bot
2. Запустите файл simple_backup.py командой:
   python simple_backup.py

3. Архив будет создан в папке C:\Dev\ с именем:
   event-bot-backup-ГГГГММДД-ЧЧММСС.zip

4. Также можно использовать встроенную функцию Windows:
   - Выделите папку event-bot
   - Правой кнопкой -> Отправить -> Сжатая ZIP-папка

Текущий путь к проекту: C:\Dev\event-bot
Архив будет создан в: C:\Dev\
